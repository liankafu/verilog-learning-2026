准备**雨骤(恩捷伦®)杯**（第十届BJT版本）的关键是**分阶段推进**，从**零硬件、低成本**起步，逐步实现从理论方案到自动化测试系统。以下基于官网任务书（http://univ.ciciec.com/nd.jsp?id=1011）、往届经验和B站/社区复盘，给出最实用、可操作的准备路径。假设你是台北学生、低预算、低年级/基础阶段，优先远程/免费资源。

### 总体准备原则
- **时间线**：初赛（纯电脑）1-2个月 → 分赛区决赛（软硬件）1-2个月 → 总决赛（现场优化）1个月。
- **团队建议**：2-4人，分工（一人电路/仿真、一人编程/GUI、一人文档/答辩）。
- **核心资源**：官网 + 钉钉群（报名后加） + B站培训视频 + 免费LTspice。
- **预算**：初赛0元；后期可选租借硬件（群里问有偿租借，几十到几百元）。

### 阶段性准备路径

#### 阶段1：基础准备（1-2周，现在就开始）
**目标**：懂题目、懂BJT参数、会用工具。

1. **读懂赛题**  
   - 官网页面：http://univ.ciciec.com/nd.jsp?id=1011（任务书PDF下载）。  
   - 重点读：背景、必测指标（线性精度 + 输出特性曲线）、国产/进口对比要求、提交材料。  
   - 打印或标注：必测参数列表（线性精度、Ic-Vce曲线、β、Vbe、VCE(sat)、ICBO/IEBO、BVCBO/BVCEO等）。

2. **复习BJT基础知识**  
   - 教材：  
     - 《模拟电子技术基础》（童诗白）第3-4章（BJT放大电路、特性曲线）。  
     - 《电子技术基础》（康华光）BJT部分。  
   - 免费PDF/在线：B站搜“BJT 特性曲线 讲解”或知乎专栏。  
   - 重点掌握：共射极配置、输入/输出特性、三大工作区、Early效应、温度影响。

3. **下载 & 熟悉工具**  
   - **仿真软件**：LTspice（免费，Analog Devices官网下载：https://www.analog.com/en/design-center/design-tools-and-calculators/ltspice-simulator.html）。  
     - 教程：B站“LTspice BJT 输出特性”或CSDN教程（搜“LTspice BJT 仿真”）。  
   - **雨珠平台准备**：报名后加钉钉群，申请免费远程账号 + 下载“仪器操场软件”/API开发包（Python/C/LabVIEW支持）。  
   - **器件模型**：LTspice内置2N2222等；国产如S8050/C9013模型网上搜“.lib”文件导入。

4. **加入渠道**  
   - 钉钉群（官网col.jsp?id=120有群号）。  
   - 微信公众号：恩捷伦中国 / 芯人类。  
   - B站关注“芯人类IC社区”（有雨骤杯培训系列）。

#### 阶段2：初赛准备（核心阶段，1-2个月）
**目标**：提交完整方案 + 仿真结果（纯电脑，无硬件）。

1. **选器件**  
   - 进口：2N2222 / 2N3904 / BC547（常见小信号NPN）。  
   - 国产：S8050 / C9013 / 9014（淘宝买样品参考参数）。  
   - 选等效型号对比。

2. **自己搭测试电路 & 仿真验证**（最重要）  
   - **线性精度**：固定VCE=5V，扫IB（0.01mA~2mA），测IC，计算线性误差（实际IC vs 理想直线偏差%）。  
   - **输出特性**：固定Ib=0.1/0.5/1/2mA，扫VCE=0~10V，画曲线族。  
   - 其他参数：β、Vbe(on)、VCE(sat)、ICBO等（逐个搭电路）。  
   - LTspice操作：  
     - 新建 → 放NPN（2N2222） → 加电压源/电流源 → .dc命令设置扫描。  
     - 教程参考：B站“LTspice BJT 输出特性曲线”或CSDN文章。  
   - 输出：曲线图 + 数据表 + 对比分析（国产曲线更平坦？饱和压降更低？）。

3. **写理论方案**  
   - 文档（Word）：  
     - 背景 + 器件选型 + 测试电路原理图（自己画的） + 仿真结果 + 国产/进口对比分析。  
     - 雨珠平台规划：怎么用API分配I/O（电压源、ADC采集）、实现自动化（Python脚本流程）。  
   - PPT（≤20页）：团队 + 市场意义 + 方案 + 结果 + 创新点。  
   - 视频（5min MP4）：录屏展示LTspice仿真过程 + 波形讲解。

4. **提交**：官网平台上传PPT + 文档 + 视频 + 仿真文件。

#### 阶段3：分赛区决赛准备（进复赛后，1-2个月）
1. **编程实现自动化**  
   - 用Python调用雨珠API（群文件有SDK）。  
   - 实现：一键采集 → 计算参数 → 画GUI曲线（Tkinter/Matplotlib）。  
   - 教程：B站“雨骤杯 培训直播”（第一/第二弹）有API实战。

2. **载板设计**（可选但加分）  
   - 用KiCad/EasyEDA画30pin/26pin DIP载板，接BJT。  
   - 可远程云测（无需自购硬件）。

3. **更新材料**：加代码、海报、测试结果。

#### 阶段4：总决赛准备（现场2小时开发）
- 优化程序：现场用雨珠Model-X + 你的代码完善自动化测试。  
- 练习：用Python框架模拟现场环境（群里有示例）。  
- 答辩：团队分工清晰，每人说清楚贡献。

### 避坑 & 高分Tips
- **初赛别抄**：电路必须自己搭，参考Datasheet但要加分析。  
- **国产对比**：量化差异（表格 + 曲线叠加），别只说“国产更好”。  
- **自动化**：GUI友好、一键运行、无干预是高分点。  
- **时间管理**：先冲初赛（仿真最稳），复赛再学API。  
- **资源**：群里问“免费账号”“培训回放”“模型文件”。
